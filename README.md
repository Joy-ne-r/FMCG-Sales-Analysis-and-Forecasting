**FMCG Sales Analysis: Trends, Drivers, and Forecasting Insights**

**Project Overview**

This project analyzes Fast-Moving Consumer Goods (FMCG) sales data to uncover market trends, demand drivers, and forecasting insights. It highlights challenges in demand volatility, promotions, stock imbalances, and regional discrepancies, while proposing strategies for inventory optimization, promotion targeting, and market prioritization.

**Objectives**

Understand market dynamics and demand drivers.

Analyze sales by country and SKU performance.

Classify SKUs by ABC and XYZ frameworks.

Correlate price, promotions, and volume.

Visualize seasonality (monthly, weekly, quarterly).

Forecast demand using statistical and machine learning models.

**Methodology**

**Data Analysis:** ABC/XYZ classification, Pareto charts, correlation matrices.

**Visualization:** Box plots, heatmaps, seasonality line graphs.

**Forecasting Models:** Linear regression and machine learning approaches.

**Key Findings**

**Country Sales:** Italy leads in net sales; Netherlands underperforms.

**SKU Insights:** Few SKUs  drive most revenue; imbalance with too many Class C SKUs.

**Volatility:** Class Y SKUs show higher unpredictability (median CV ~0.53).

**Correlation:** Discounts and promotions strongly correlated (0.92); weak link between list price and sales.

**Seasonality:** Mid-year (May–July) and late-year (Oct–Dec) spikes; weekend sales higher.

**Forecasting Results**

Linear Regression achieved R² = 0.70, indicating reasonably strong predictive power.

Forecasts generated for 10 SKUs with varied demand profiles.

**Strategic Recommendations**

**Inventory Optimization:** Reduce Class C overstocks; increase Class A availability.

**Promotion Strategy:** Focus discounts/promotions on top-revenue SKUs (Class A/B).

**Market Prioritization:** Strengthen Italy and Spain; reassess Netherlands.
